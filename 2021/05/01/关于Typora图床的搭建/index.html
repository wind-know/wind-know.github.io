# 背景

Markdown 是一门易于上手能帮助作者专心写作的文档编辑语言

typora 是一款优雅的 Markdown 编辑器，为了解决图片在上传平台无法显示，而复制链接显得过于繁琐，特搭建本地图床（万幸 typora

支持此功能）
